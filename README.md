# Budget Tracker

A personal budget tracking application built with Next.js. Track income and expenses, view your balance at a glance, and categorize transactions to understand where your money is going.

## About

Budget Tracker provides a clean, dark-themed dashboard for managing your finances. The app displays a summary of your current balance, total income, and total expenses, alongside a list of recent transactions. Users can add new transactions through a modal form, specifying a name, amount, and category. Transactions are automatically categorized as income or expense based on the selected category.

The interface is fully responsive and works across desktop and mobile devices. On smaller screens, the layout adapts with stacked summary cards and a bottom-sheet modal for adding transactions.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- CSS Modules
